This directory includes the Bot Libre SDK for iOS iPhone, iPad, and other iOS devices.

To use this in your own iOS app, just copy and import the code to your own project structure.
You can either just use the LibreSDKConnection API, or use, or customize the ui classes for your content.

To create a new iOS app you will need to download the Xcode IDE from Apple.

https://developer.apple.com/xcode/

You can publish iOS apps on iTunes Connect.

https://itunesconnect.apple.com
